Security Hardening

Top Notch measures securing our servers.

UFW: Uncomplicated Firewall, is a front-end to iptables. manages the firewall in an easy-to-use interface. It's well-supported and popular in the Linux community. we use UFW to block all incoming and outgoing ports by default and open only service needed ports.

OpenVPN : Our Staging Servers are secured by OPENVPN. which will grant access to authenticated users only.

Vertical scaling will not work when there is lot of data and traffic to deal with. There comes the need for horizontal scaling with Load Balancing. It is a technique of distributing your requests to multiple web servers. We master in Amazon Web Services Load Balancer.
